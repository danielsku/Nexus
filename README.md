

# Nexus

<img width="960" height="720" alt="Nexus_Logo" src="https://github.com/user-attachments/assets/08d4b186-58ea-4a95-956a-3089b7b51016" />

With digital privacy under constant threat, Nexus is a secure chat application enabling multi-user communication with account creation and offline message storage, ensuring no data is shared with third parties. Developed using Java Spring Boot (MVC), RESTful APIs, Hibernate/JPA, WebSockets (STOMP), and JavaFX, the project emphasizes real-time messaging, secure authentication, and scalable cloud-ready deployment.
